## 🌿 BloomWell – Smart Wellness Tracking System

BloomWell is a desktop-based wellness application developed using **Scala, ScalaFX, and MVC architecture**. The system is designed to help users monitor and improve their daily well-being by tracking **mood, sleep patterns, and stress levels**.

The application allows users to log multiple mood entries per day, record sleep data, and visualize insights through interactive charts. It also includes a **wellness scoring system**, **stress level analysis**, and a **badge-based achievement system** to encourage consistent healthy habits.

Built with a strong focus on **object-oriented design principles**, BloomWell demonstrates the use of **inheritance, abstraction, encapsulation, traits, and composition**, along with database integration using **ScalikeJDBC and Apache Derby**.

### ✨ Key Features
- 📊 Monthly mood and sleep visualization (charts)
- 😊 Multiple mood tracking per day
- 😴 Sleep tracking with duration and quality
- 🧠 Daily stress level analysis
- 🏆 Achievement and badge system
- 🔐 User authentication and profile management

BloomWell aims to provide users with meaningful insights into their lifestyle while reinforcing good habits through data-driven feedback.
